# Signed-Out Screen Design Reference

## Source of Truth

- Final desktop and mobile mockups define layout and composition.
- Real application screenshots define carousel content.
- The current Daily Planner app defines colors, typography, logo, controls, and theme behavior.
- Do not reproduce Gemini-invented phone frames, navigation, sparkles, tasks, or interface elements.

## Approved Direction

- Authentication content first and visually dominant.
- Desktop uses a two-column layout.
- Mobile stacks authentication above the carousel.
- Carousel uses real Daily Planner screenshots.
- One active slide with arrows and navigation dots.
- Respect reduced-motion preferences.
